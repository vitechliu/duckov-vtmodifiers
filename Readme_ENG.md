Loot boxes, enemy equipment, crafted equipment, etc. will now be randomly attached with Modifier! Start Farming!

Update 0.4.0
- Now supports "Reforging" and "Forging", which can be operated by right clicking on the equipment in the base. Forging(Adding Modifier to non-modifiered items) costs much more..
- Next version will support feature of mod configuration, etc.

Update 0.3.0
- Now modifiers have been added to more types of equipment (including armor, helmets, backpacks, etc.)!
- Removed or lowered a large number of negative attributes;
- Fixed some of the modified properties with excessive decimal places and resolved some UI display issues;

Update 0.2.1
- The attribute bonus generated by modifiers now will fluctuate from 50% to 100% maximum (except for some special modifiers)

- Note:
