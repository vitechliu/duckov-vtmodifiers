更新日志
### 0.5.1
- 现在部分兼容mod:Scav模式，在Scav模式中敌人掉落的装备也会有词缀。**请确保Scav模式的加载位置在本mod之前！**
- 更新ModSetting API

### 0.5.0
- 现在近战武器也支持了词缀！
- 加入了"词缀属性是否固定"开关，开启后词缀属性将固定为最大值
- 修复了基地里假人身上的护甲也会有随机词缀的问题（怪不得测试穿甲会不准）
- 现在兼容[ModSetting](https://steamcommunity.com/sharedfiles/filedetails/?id=3595729494)模组，可以更方便的修改参数
- 对于出现重复词缀显示的情况（如 精准 精准 AK47)，加了补丁以避免，但请注意这不是bug，如果你出现这个问题，代表你的mod打太多了！有的mod导致了重复加载问题

### 0.4.3
- 所有对提升护甲的数值减少一半
- 配置菜单新增对护甲、伤害数值的倍率调整，觉得太强/太弱的可以手动调整，[b]需重启游戏生效！[/b]
- 修复了部分原版显示保留两位小数变成一位的情况（图腾）

### 0.4.2
- 现在按Ctrl+F3可以打开模组配置菜单
- 修复了会导致物品属性展示负面词缀红色变成白色的问题，同时现在数值部分不会变成紫色。
- 修复了在联机MOD中，客机会无限子弹无限射速的问题！
- 修复了部分词缀缺失词条的问题；

注意：
1. 合成/物资箱的 装备初始带有词缀的概率为75%，敌人武器为40%
2. 因为敌人（甚至boss）生成时，也可能会带有强力武器，或者小概率特别硬的护甲，小心被秒杀！
3. 如果有bug，或者和其他热门mod有冲突请尽量在讨论区而非留言反馈(留言不能回复和跟进问题)！
4. 觉得不错的鸭鸭们点个好评！感谢大家~
